# Substack_InlineLatex_Extension
I built a substack inline latex extension that uses an already existing latex_inline node from Substack and used JS to detect when a user types $...$ to render as inline latex. 

### Google Chrome Extension
The config file is formatted to be uploaded on chrome. To use it: Enable developer mode. Then upload the substack_latex_extension folder onto your chrome extensions tab. Ensure that the extension is enabled / turned on and you are ready to go! 

#### Privacy Policy for Chrome
1. Introduction: This is the privacy policy page for the Substack Latex Chrome Extension. 
2. Data We Collect: We do not collect any user data for any purpose. 
3. How We Use Your Data: We do not collect any user data so we will not use any user data for any functionality. 
4. Third-Party Data Sharing: The extension is built on top of Substack. Any data collected by Substack is not shared or transfered or extracted from Substack by this extension.
5. Google Analytics: This extension has opted in for Google Analytics which tells me how many users have downloaded and used the extension.
